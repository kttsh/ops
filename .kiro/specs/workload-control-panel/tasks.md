# Implementation Plan

- [x] 1. shadcn/ui Popover コンポーネントの導入
  - shadcn CLI で Popover コンポーネントを追加し、`@radix-ui/react-popover` を依存関係に含める
  - 生成された Popover コンポーネントが正しくインポート・使用可能であることを確認する
  - _Requirements: 3.2, 4.2, 5.1_

- [x] 2. ColorPickerPopover コンポーネントの作成
- [x] 2.1 (P) 色選択 Popover コンポーネントを新規作成する
  - 選択中の色を丸型スウォッチとして表示するトリガーボタンを実装する
  - クリックで Popover を開き、渡された色パレット全体をグリッド表示する
  - Popover 内で色を選択すると onChange コールバックを発火し、Popover を閉じる
  - Popover 外クリックで色を変更せずに閉じる（Radix 標準動作）
  - 現在選択中の色をリングで視覚的に区別する
  - 色パレット配列、現在の色、onChange コールバックの 3 つの props を受け取る
