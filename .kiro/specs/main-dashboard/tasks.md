- [ ] 3. チャートコンポーネントの実装
- [ ] 3.1 API レスポンスをチャート用データに変換するカスタムフックを実装する
  - 集約チャートデータから月別フラットデータ配列への変換ロジックを実装する
  - 案件タイプ別のエリアシリーズ設定を表示順序の昇順で生成する
  - 間接作業ケース別のエリアシリーズ設定をグレー系カラーで生成する
  - キャパシティシナリオ別のラインシリーズ設定を青系グラデーションで生成する
  - 凡例パネル用の月別詳細データを Map 構造で構築する
  - メモ化でデータ変換結果を安定化する
  - _Requirements: 1, 2, 18, 22_

- [ ] 3.2 積み上げエリアチャート + キャパシティライン描画コンポーネントを実装する
  - Recharts の複合チャートで案件タイプ別の積み上げエリア（不透明度 0.8）と間接作業エリア（不透明度 0.7）を描画する
  - スタック順序を「下から順に: 間接作業 → 案件工数（案件タイプ順）」に設定する
  - キャパシティシナリオごとに破線スタイルの折れ線グラフを重畳表示する
  - X 軸に年月（YYYY/MM）、Y 軸に工数（人時）を表示する
  - 低スペック PC 対応: アニメーション全面禁止、ドット描画禁止、アクセシビリティレイヤー無効化を設定する
  - カーブタイプを monotone に統一し、チャートコンテナに CSS contain を設定する
  - コンポーネントをメモ化し、requestAnimationFrame によるマウスムーブスロットリングを実装する
  - レスポンシブコンテナでチャート幅をコンテナに追従させる
  - _Requirements: 1, 2, 20_

- [ ] 3.3 チャートクリック・ホバーイベントのハンドリングを実装する
  - チャートエリアクリック時に凡例パネルへ月データの固定表示をディスパッチする
  - マウスムーブ時にホバー中の月のデータを凡例パネルに一時表示する
  - マウスリーブ時に初期表示（最新月）に戻す
  - 縦線カーソル（破線）で現在ホバー中の月を示し、ツールチップボックスを非表示にする
  - イベントハンドラをメモ化で安定化する

  - _Contracts: useLegendState API_

- [ ] 3.4 チャート上のコンテキスト操作を実装する

  - _Requirements: 17_

  - ホバー、ホバー離脱、クリック、ピン解除、ドリルダウン切替の各アクションを処理する

  - ドリルダウン展開は固定状態でのみ有効とする
  - ページロード時に最新月のデータを初期表示する

- [ ] 4.2 固定凡例パネル UI コンポーネントを実装する

  - 間接作業セクション（色ラベル + ケース名 + 工数値）を表示する
  - キャパシティセクション（色ラベル + シナリオ名 + 値）を表示する
  - サマリー（合計工数・稼働率）を表示する

- [ ] 4.3 案件タイプドリルダウン機能を実装する
