  - リソース作成後の Location ヘッダーをベースパスとリソース ID から設定する関数を実装する
  - 両関数のユニットテスト（レスポンス構造、ページ算出、空データ、ヘッダー値の検証）を作成する

- [x] 2. (P) 既存スキーマバリアントの差分調査
