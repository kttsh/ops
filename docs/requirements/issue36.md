# issue list

## 画面全般
・カードUIのマウスオーバー時の影は不要。
・画面切り替え時のアニメーションは禁止。低スペックPCで快適に表示されるようにしたい。

## 対象画面　/workload
・BUをビジネスユニット、に変名
・全選択の文字の右側にビジネスユニットのチェックボックスを並べて、1行表示とする。
・チャートの下に余白ができてしまっているので、チャートの高さと固定凡例の高さを揃える。
・チャートとテーブルの両方のビューで縦方向のスクロールが効かないので、テーブルが上部の一部しか見えない。スクロールできるようにしたい。
・チャートだけの表示、テーブルだけの表示のときは、縦スクロールはしない。
・チャートだけの表示のとき、チャートの高さはカード内いっぱいに表示したい。現状は、チャート、固定凡例の下に余白ができている。
・チャートエリアをクリックした際の、チャートの外側の黒線を消したい。何も線を付けない。

## 対象画面　チャート
・Areaの境界に線は不要
・1つのAreaは単一の色で塗る。グラデーションは禁止。

## 対象画面 コントロールパネル
・コントロールパネルのタブ名を「設定」から「チャート設定」に変更する。
・設定＞案件設定　の縦スクロールバーは不要。コントロールパネル全体の縦スクロールと2重になってしまっているので。
・色選択は、どの色が選択されているか、ぱっとわかりづらい。色を並べておくのではなく、画面上には選択した色を表示しておき、色をクリックすると色の選択肢があり、変更する仕様にする。


## 対象画面　master/work-types
・作業種類マスタでカラーの選択は不要。カラーはworkload画面のコントロールパネルで選択するため。

## 対象画面　master/* マスター管理画面全て（案件以外）
・一覧画面で各レコードを選択した際、現状は詳細画面に遷移し、さらに編集をクリックすると編集画面に遷移する。新規登録の際も/new画面に遷移する。すべて一覧画面内で完結できるように、詳細をSheet表示し、編集をすると、Sheet内のフィールドがEditableになる仕様にする。

## サイドメニュー
・間接作業・キャパシティを選択した際、「間接作業・キャパシ」の後に改行が入ってしまっている。少しサイドメニューの幅を広げてほしい。
